Objetivo de SF.2:
Crear la capa de datos para analizar una partida PGN completa, sin hacer todavía el diseño frontend final.

No queremos máxima profundidad ni máximo Elo.
ChessLens necesita análisis rápido, claro, friendly y que no bloquee la app.

Principios de producto:

* La evaluación se mostrará siempre desde el punto de vista de las blancas.
* Positivo = blancas mejor.
* Negativo = negras mejor.
* Ejemplo:

  * +1.1 = blancas tienen aproximadamente 1 peón y 10 centipeones de ventaja.
  * -1.1 = negras tienen aproximadamente 1 peón y 10 centipeones de ventaja.
* Por defecto queremos las 2 mejores jugadas.
* Más adelante habrá configuración para 1, 2 o 3 mejores jugadas.

Tarea técnica:
Crear una capa que:

1. Reciba un PGN válido.
2. Use chess.js para recorrer la partida.
3. Genere una lista de posiciones/FEN antes y después de cada jugada.
4. Llame al wrapper Stockfish ya creado.
5. Use MultiPV = 2 por defecto.
6. Normalice siempre la evaluación a perspectiva blanca.
7. Devuelva resultados estructurados por jugada.

Formato sugerido de resultado:

type AnalysisLine = {
move: string;
san?: string;
scoreCpWhite?: number;
mateWhite?: number;
bestMoveUci?: string;
pv?: string[];
};

type MoveAnalysis = {
ply: number;
moveNumber: number;
side: "w" | "b";
san: string;
fenBefore: string;
fenAfter: string;
evalBeforeCpWhite?: number;
evalAfterCpWhite?: number;
evalLossCp?: number;
bestLinesBefore: AnalysisLine[];
bestLinesAfter?: AnalysisLine[];
label?: "excellent" | "good" | "inaccuracy" | "mistake" | "blunder" | "critical";
};

Crear preferiblemente:

* client/src/lib/pgn-analysis.ts
* client/src/hooks/use-stockfish-analysis.ts

O nombres equivalentes si encajan mejor con el proyecto.

Importante sobre la evaluación:
Si Stockfish devuelve scoreCp desde el punto de vista del bando que mueve, convertirlo a perspectiva blanca:

* si sideToMove en la FEN es "w", scoreCpWhite = rawScoreCp;
* si sideToMove en la FEN es "b", scoreCpWhite = -rawScoreCp.

El análisis debe ser bajo demanda.
No debe arrancar automáticamente al cargar una partida.
